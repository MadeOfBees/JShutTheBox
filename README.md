# JShutTheBox

a simple javascript port of the game Shut The Box, made with materialUI and react.

## "Installation"/Instructions:

Click the Github Pages link: https://madeofbees.github.io/JShutTheBox/ (when version 1 releases) to navigate to the site then once there you can play Shut The Box, if you don't know the rules click the "How To Play" button in the bottom left corner. if you want to toggle dark mode click the "Dark Mode" button in the bottom right corner.
